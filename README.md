# Mafia Telegram Bot

Бот для создания и записи на игры по спортивной мафии ВМК.

## Что реализовано

- Регистрация пользователя:
  - запрос контакта (номер телефона);
  - ввод никнейма;
  - проверка уникальности никнейма.
- Пользовательское меню:
  - `Расписание игр` -> список кнопок по количеству игр;
  - выбор игры;
  - запись на роль:
    - ведущий (максимум 1),
    - судья (максимум 3),
    - игрок (максимум 10).
- Админ-авторизация:
  - админ автоматически определяется по номеру телефона из `.env` (`ADMIN_PHONE`);
  - после регистрации с этим номером доступны кнопки `Админ-меню` и `Расписание игр`.
- Админ-меню:
  - добавление админа по Telegram ID;
  - удаление админа по Telegram ID;
  - создание игры (дата/время + место).

## Запуск

1. Установить Python 3.11+.
2. Установить зависимости:

   ```bash
   pip install -r requirements.txt
   ```

3. Создать `.env` на основе примера:

   ```bash
   cp .env.example .env
   ```

4. Заполнить переменные в `.env`:
   - `BOT_TOKEN` — токен бота от BotFather;
  - `ADMIN_PHONE` — номер телефона администратора;
   - `SUPERADMIN_IDS` — список Telegram ID через запятую (эти пользователи автоматически админы);
   - `DB_PATH` — путь до SQLite базы.

5. Запустить:

   ```bash
   python bot.py
   ```
